HTML HOME

HTML is a markup language that defines the **structure** and **meaning** of web pages. CSS/JS build upon HTML.

<!DOCTYPE html>
      <html lang="en">
      <meta charset="utf-8">
      <title>My First Page</title>
      <h1>Hello World</h1>
      <p>HTML is the blueprint of the web.</p>
DOMBasicsBoilerplate

Introduction

Browsers parse HTML to create a DOM tree, then apply CSS to render a visual tree.

  • A **semantic** structure is good for SEO, accessibility, and maintenance.
  • The **standard** is updated via the Living Standard (WHATWG).
  • Check **compatibility** on MDN or Can I use.

Basic

  • <!DOCTYPE html>: HTML5 declaration.
  • lang, charset, and viewport are essential.
<!DOCTYPE html>
      <meta charset="utf-8">
      <h1>Basic Structure</h1>
      <p>Even simple HTML renders here.</p>

Elements

Element = start tag + content + end tag. Void elements have no end tag (e.g., <img>).

Block vs Inline

  • **Block**: Takes up the full width (e.g., div, p, h1).
  • **Inline**: Takes up only the space it needs (e.g., span, a, img).

id vs class

<div id="hero">Unique, one per page</div>
      <div class="tag">Common to multiple elements</div>
