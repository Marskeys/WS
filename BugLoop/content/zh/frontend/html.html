HTML 主页

HTML是一种定义网页结构和意义的标记语言。CSS/JS则在其基础上工作。

<!DOCTYPE html>
      <html lang="zh">
      <meta charset="utf-8">
      <title>我的第一页</title>
      <h1>你好</h1>
      <p>HTML是网页的蓝图。</p>

DOM基础样板代码

简介

浏览器解析HTML以构建DOM树，然后应用CSS来渲染渲染树。

  • 语义化结构有利于SEO/可访问性/维护。
  • 标准遵循Living Standard (WHATWG) 更新。
  • 兼容性可在MDN/Can I use上查询。

基础

  • <!DOCTYPE html>: HTML5声明。
  • lang, charset, viewport是必须的。

元素

元素 = 开始标签 + 内容 + 结束标签。空元素没有结束标签 (如 <img>)。

表单

链接 label[for] ↔ input[id] 对于可访问性至关重要。

语义化

<header>页眉</header>
      <nav>导航</nav>
      <main><article>文章</article></main>

      <footer>页脚</footer>

块级 vs 内联

  • 块级 (Block): 占据整行 (例如：div, p, h1)
  • 内联 (Inline): 只占据所需空间 (例如：span, a, img)

id vs class

<div id="hero">唯一的，每页一个</div>
      <div class="tag">多个元素共用</div>

<head> & Meta

<meta name="description" content="摘要">
      <meta property="og:title" content="社交媒体标题">
      <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
