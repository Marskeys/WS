HTML 완전정복: 구조부터 실전 활용까지

HTML(HyperText Markup Language)은 웹의 기초 설계도다.
CSS가 외형(색, 크기, 배치), JavaScript가 동작(이벤트, 데이터 처리)을 맡는다면, HTML은 그 위에 얹히는 모든 요소의 위치·역할·의미를 정의한다.
브라우저는 HTML 문서를 읽어 DOM(Document Object Model)이라는 트리 구조로 변환한 뒤, CSS와 합쳐 화면에 렌더링한다.

1. HTML의 본질

  • 구조화: 제목, 본문, 이미지, 표 등을 논리적 계층으로 나눈다.
  • 의미 부여: 시맨틱 태그를 사용해 콘텐츠의 목적을 명확히 한다.
  • 브라우저와의 약속: HTML 태그와 속성은 브라우저가 해석하는 표준 규칙을 따른다.

1.1 기본 문서 구조

<!DOCTYPE html>
  <html lang="ko">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>문서 제목</title>
    </head>
    <body>
      <h1>안녕하세요</h1>
      <p>이것은 HTML 예제입니다.</p>
    </body>
  </html>

TIP: <!DOCTYPE html>은 HTML5 표준 문서임을 선언하는 필수 문구다.

2. 시맨틱 태그와 의미

시맨틱 태그는 단순히 화면에 보여주기 위한 게 아니라, 콘텐츠의 목적과 역할을 검색엔진·보조기기에 전달한다.

  • <header> - 페이지나 섹션의 머리말
  • <nav> - 내비게이션 링크 집합
  • <main> - 문서의 핵심 콘텐츠(한 번만 사용)
  • <section> - 주제별 구획
  • <article> - 독립적인 콘텐츠 블록
  • <aside> - 부가 정보
  • <footer> - 꼬리말

3. 텍스트와 링크

<h1>제목</h1>
  <p>단락 내용.</p>
  <a href="https://example.com" target="_blank" rel="noopener">외부 링크</a>

TIP: 외부 링크에 rel="noopener"를 붙이면 보안과 성능이 향상된다.

4. 이미지와 미디어

<img src="cat.jpg" alt="고양이 사진" width="400">
  
<video controls width="400">
    <source src="movie.mp4" type="video/mp4">
    브라우저가 video 태그를 지원하지 않습니다.
  </video>

TIP: alt 속성은 시각장애인용 스크린 리더와 SEO 모두에 중요하다.

5. 폼 요소

<form action="/submit" method="post">
    <label for="email">이메일</label>
    <input type="email" id="email" name="email" required>
    <button type="submit">제출</button>
  </form>

6. 테이블

<table>
    <caption>월별 판매량</caption>
    <thead>
      <tr><th>월</th><th>판매량</th></tr>
    </thead>
    <tbody>
      <tr><td>1월</td><td>100</td></tr>
      <tr><td>2월</td><td>150</td></tr>
    </tbody>
  </table>

7. 접근성(A11y) 원칙

  • 이미지에 대체 텍스트 제공
  • 폼 요소와 <label> 연결
  • 헤딩 계층 구조 유지
  • 색상만으로 정보 전달하지 않기

8. SEO 최적화

  • <title>과 <meta name="description">를 페이지별로 정확히 작성
  • 핵심 콘텐츠를 <main>에 배치
  • 의미 있는 URL 구조 사용

9. 성능 최적화

  • 이미지 용량 최소화 + loading="lazy"
  • 스크립트에 defer 또는 async

10. 실수 방지
