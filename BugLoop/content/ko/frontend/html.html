HTML 기초 및 활용 튜토리얼

본 문서는 웹 개발의 기초를 이루는 HTML(HyperText Markup Language)의 개념, 구조, 시맨틱 태그, 접근성, SEO, 성능 최적화 등에 대하여 체계적으로 설명한다. HTML은 웹 페이지의 구조를 정의하며, CSS와 JavaScript와 함께 웹 프론트엔드의 3대 핵심 기술로 분류된다.

1. HTML의 개요

HTML은 웹 브라우저가 이해할 수 있는 마크업 언어로, 콘텐츠의 의미와 구조를 지정한다. 모든 HTML 문서는 <html> 요소를 루트로 하여 <head>와 <body>로 구분된다.

1.1 기본 구조

<!DOCTYPE html>
  <html lang="ko">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>문서 제목</title>
    </head>
    <body>
      <h1>안녕하세요</h1>
      <p>이것은 HTML 예제입니다.</p>
    </body>
  </html>

2. 시맨틱 태그

시맨틱 태그는 요소의 의미를 명확히 하여 검색엔진 최적화와 접근성을 높인다.

  • <header> - 페이지나 섹션의 머리말.
  • <nav> - 내비게이션 링크 집합.
  • <main> - 문서의 주 콘텐츠(한 페이지에 한 번만 사용).
  • <section> - 주제별 구획.
  • <article> - 독립적인 콘텐츠 블록(뉴스, 블로그 글 등).
  • <aside> - 부가 정보.
  • <footer> - 꼬리말.

3. 텍스트 및 링크

<h1>제목</h1>
  <p>단락 내용.</p>
  <a href="https://example.com" target="_blank" rel="noopener">외부 링크</a>

4. 이미지 및 미디어

이미지는 alt 속성을 통해 대체 텍스트를 반드시 제공해야 한다.

<img src="cat.jpg" alt="고양이 사진" width="400">

비디오 예시:

<video controls width="400">
    <source src="movie.mp4" type="video/mp4">
    브라우저가 video 태그를 지원하지 않습니다.
  </video>

5. 폼 요소

<form action="/submit" method="post">
    <label for="email">이메일</label>
    <input type="email" id="email" name="email" required>
    <button type="submit">제출</button>
  </form>

6. 테이블

<table>
    <caption>월별 판매량</caption>
    <thead>
      <tr><th>월</th><th>판매량</th></tr>
    </thead>
    <tbody>
      <tr><td>1월</td><td>100</td></tr>
      <tr><td>2월</td><td>150</td></tr>
    </tbody>
  </table>

7. 접근성(A11y) 원칙

  • 모든 이미지에 대체 텍스트 제공.
  • 폼 요소와 <label> 연결.
  • 헤딩 계층 구조 유지.
  • 색상만으로 정보 전달하지 않음.

8. SEO(Search Engine Optimization)

  • <title>과 <meta name="description"> 정확히 작성.
  • 핵심 콘텐츠를 <main>에 배치.
  • 의미 있는 URL 구조 사용.

9. 성능 최적화

  • 이미지 용량 최소화 및 loading="lazy" 속성 활용.
  • 스크립트에 defer 또는 async 적용.

10. 실수 방지

  • <div> 남용 대신 시맨틱 태그 활용.
  • 잘못된 속성 사용 방지.
