HTML HOME

HTML은 웹 페이지의 구조와 의미를 정의하는 마크업 언어다. CSS/JS는 HTML 위에 올라탄다.

<!DOCTYPE html>
  <html lang="ko">
  <meta charset="utf-8">
  <title>첫 페이지</title>
  <h1>안녕하세요</h1>
  <p>HTML은 웹의 설계도입니다.</p>
Try it
DOM기본보일러플레이트

Introduction

브라우저는 HTML을 파싱하여 DOM 트리를 만들고, CSS를 적용해 렌더 트리로 그린다.

  • 시맨틱 구조는 SEO/접근성/유지보수에 유리
  • 표준은 Living Standard(WHATWG)에 따라 업데이트
  • 호환성은 MDN/Can I use로 확인

Basic

  • <!DOCTYPE html>: HTML5 선언
  • lang, charset, viewport는 필수
Try it

Elements

요소 = 시작 태그 + 내용 + 종료 태그. 빈 요소는 종료 태그 없음(<img> 등).

<p>문단</p>
  <img src="/path/cat.jpg" alt="고양이">

Attributes

<a href="/docs" target="_blank" rel="noopener">문서</a>
  <input type="checkbox" checked> 불리언 속성은 값 없이 존재=참

Headings

<h1>대제목</h1>
  <h2>중제목</h2>
  <h3>소제목</h3>

Paragraphs

<p>첫 문단</p>
  <p>둘째 문단</p>
  <hr>
  <p>셋째<br>줄바꿈</p>

Styles

<p style="color:tomato; font-weight:bold">경고</p>

인라인 스타일은 빠르지만 유지보수성이 낮음 → CSS 권장

Links

<a href="https://example.com" target="_blank" rel="noopener">외부</a>
  <a href="#blt-tables">같은 문서 내 이동</a>

Images

<img src="/img/photo.jpg" alt="행복한 강아지" width="420">

alt는 접근성과 SEO 핵심

Lists

<ul><li>불릿</li><li>리스트</li></ul>
  <ol><li>순서</li><li>리스트</li></ol>
  <dl><dt>용어</dt><dd>설명</dd></dl>

Tables

<table>
    <caption>월별 판매량</caption>
    <thead><tr><th>월</th><th>판매량</th></tr></thead>
    <tbody>
      <tr><td>1월</td><td>100</td></tr>
      <tr><td>2월</td><td>150</td></tr>
    </tbody>
  </table>

Forms

<form action="/submit" method="post">
    <label for="email">이메일</label>
    <input id="email" name="email" type="email" required>
    <button type="submit">제출</button>
  </form>

label[for] ↔ input[id] 연결은 접근성 필수

Semantics

<header>머리말</header>
  <nav>내비</nav>
  <main><article>글</article></main>
  <aside>보조</aside>
  <footer>꼬리말</footer>

Block vs Inline

  • Block: 한 줄 전체 차지 (예: div, p, h1)
  • Inline: 내용만큼 (예: span, a, img)

id vs class

<div id="hero">고유 1개</div>
  <div class="tag">여러 요소에 공통</div>

<head> & Meta

<meta name="description" content="요약">
  <meta property="og:title" content="SNS 제목">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

File Paths

./img/cat.jpg    (현재 폴더)
  ../assets/a.js  (상위 폴더)
  https://cdn.example.com/lib.css (절대)
